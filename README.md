## Getting Started

```bash
# Clone the repo
git clone https://github.com/1ador/imlador.git

cd imlador

# Install dependencies
npm install

# Run locally
npm run dev

# Clean previous gh-pages contents (optional but recommended)
npx gh-pages-clean

# Build the project
npm run build

# Deploy to gh-pages branch
npm run deploy
